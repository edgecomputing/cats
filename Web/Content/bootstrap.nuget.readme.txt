
Content:
--------

The bootstrap nuget package installs content under:
/Content/bootstrap/
/Content/google-code-prettify/

Quick start: 
------------

1. Install a .less tool, such as the dotless Nuget package
2. Add the style to a page: 

    <link href="/Content/bootstrap/less/bootstrap.less" rel="Stylesheet" />

	or

    <link href="@Url.Content("~/Content/bootstrap/less/bootstrap.less")" rel="Stylesheet" />

3. View the page
